## Shopping Cart Api

## Description

Rest api developed using Nest, typeorm

## Create and start containers

```bash
$ docker-compose up api
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Appi documentation
http://localhost:3001/documentation/
